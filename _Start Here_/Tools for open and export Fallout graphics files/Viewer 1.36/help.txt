Прога "Viewer for FALLOUT2 v1.3"
Copyright (c) 24/10/2003 jab
uses Delphi5 VCL and KOL

============
Цель:
Просмотр графических ресурсов игры Fallout (и не только).
По возможности сохранение в привычные нам форматы.
============

============
Проверено в:
Windows9x 
W2k/XP
Особенностью программы является то, что нет
никаких особенностей! Программе не требуются
DirectX и прочие заморочки.
============

============
Установка программы проста!
Поместите программу и все файлы в папку (или в любую другую)
C:\Program Files\viewer\
а потом
:) просто запустите viewer.reg (кликнув 2 раза на нём)
!!!Это нужно для привязки FRM, RIX, MVE файлов к программе!!!
Если этого не делать работать всё равно будет,
но лучше привязать, так быстрее открывать :) !
============

============
Она позволяет просматривать:
*.frm *.fr0 *.fr1 *.fr2 *.fr3 *.fr4 *.fr5 - графические файлы Fallout 2
*.mve - Interplay video
*.bmp - пояснений не требует
*.ico - иконки файлов
*.gif - CompuServe GIF Images

В комплекте идут плагины:
imgpack.dll
	Silicon Graphic Images (*.bw, *.rgb, *.rgba, *.sgi)
	Autodesk Images (*.cel, *.pic)
	TIFF Images (*.tif, *.tiff)
	Enhanced PostScript images (*.eps)
	Targa Images (*.tga; *.vst; *.icb; *.vda; *.win)
	Kodak Photo CD Images (*.pcd)
	Portable Map Graphic Images (*.ppm, *.pgm, *.pbm)
	RLA Images (*.rla, *.rpf)
	Photoshop Images (*.psd, *.pdd)
	Portable Network Graphic Images (*.png)
	ZSoft PCX Images (*.pcx; *.dcx, *.pcc; *.scr)

dat.dll
	*.dat - картинки в сэйвах игры Fallout 1-2

rix.dll
	*.rix - splash from fallout

jpeg.dll
	*.jpg - JPEG

pcx.dll
	ZSoft PCX Images (*.pcx)

wal.dll
	Quake and Daikatana wal files (*.wal)
============

============
Она позволяет сохранять в:
*.bmp
*.rix
*.frm
*.jpg

... а также сохранять в то что поддерживают плагины ...
============

============
Комбинации клавиш:
Alt+F           -Menu File
Alt+F+O         -Open File
Alt+F+S         -Save File
Alt+F+E         -Exit

Alt+Shift+<     -Back Image
Alt+Shift+>     -Next Image

Alt+Z           -Menu Zoom
Alt+Z+I         -Zoom In
Alt+Z+O         -Zoom Out

Alt+P           -Menu Palette
Alt+P+L         -Load Palette (загрузка левой палитры)
Alt+P+D         -Default Palette (загрузка обычной палитры)

Alt+L           -Menu Plugins
Alt+L+S         -Plugins Select (выбор и загрузка плугинов)
Alt+L+R         -Plugins Run (выполнение выбраного плагина) 

Alt+W           -Menu Window
Alt+W+F         -Window Full Screen
Alt+W+N         -Window Normal
Alt+W+b         -Window bgColor

Alt+H           -Menu Help
Alt+H+A         -About

Для быстрого перехода к следующей картинке используйте пробел.
Для быстрого перехода к предыдущей картинке используйте <- .
============

============
Если картинка по размерам превосходит максимальный размер окна,
а посмотреть очень хотца что там, то нажатием мыши на картинке
мы переводим имэйдж в состояние alNone и зажав мышкой можем
двигать картинку по экрану(вроде не глючит). Однако просто нажав
на картинку, картинка установится в 0 по x и y координатам и на
форме появятся скрол бары, которые, посредством их передвижения,
позволят просмотреть изображение целиком.

Zoom In - растягивает(сжимает) картинку по всей активной площаде
формы. Используя эту опцию можно увеличить маленькие картинки,
либо уменьшить большие(уложить в пределы формы)
Zoom Out - Востанавливает предыдущее состояние.
